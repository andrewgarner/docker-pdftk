# docker-pdftk

Run [PDFtk Server][pdftk-server] using [Docker].

## Prerequisites

* [Docker]
* [Docker Compose][docker-compose]

## Usage

```sh
$ docker-compose run pdftk <input PDF files | - | PROMPT>
```
